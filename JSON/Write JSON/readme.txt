First, install Express: npm install express

Create an empty file named formData.json in your project directory.

Running the Application
Save the three files (index.html, client.js, server.js, and formData.json) in the same directory.
Run the server using the command: node server.js
Open your browser and navigate to http://localhost:3000.
Fill and submit the form, and the data will be appended to formData.json
